# Superstore SQL Analysis

This project focuses on data validation, data cleaning, and exploratory analysis using SQL on a retail sales dataset.

## Key Objectives
- Validate data integrity (row counts, null checks, uniqueness)
- Identify and handle true duplicate records
- Analyze sales, profit, and profit margins
- Evaluate the impact of discounts on profitability
- Derive business-focused insights and recommendations

## Tools Used
- SQL Server

## Dataset
Sample Superstore dataset (public retail dataset)

## Key Insights
- Overall profit margin is approximately 12.5%
- Technology and Office Supplies have strong profit margins (~17%)
- Furniture shows low profitability despite high sales
- Heavy discounting (30% and above) leads to consistent losses

# Power BI Dashboard

- Built an interactive dashboard to analyze sales, profit margins, and discount impact
- Key focus on category-level profitability and loss-driving discounts
- Tools: Power BI, SQL Server
